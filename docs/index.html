How the Light Tracker Works

Econ Light Monitor uses NASA nighttime lights data to estimate shifts in economic activity over time. It smooths the light-density signal, compares each region with its recent and year-ago levels, and produces a trend label for fast interpretation.

This is best used as a directional economic indicator, not as a direct GDP replacement. It is especially useful for spotting local momentum, regional divergence, and early changes in growth patterns.
