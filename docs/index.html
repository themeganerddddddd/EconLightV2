This dashboard uses NASA Black Marble monthly nighttime lights to build a public signal for regional economic momentum. It smooths the region-level signal, limits unstable percentage jumps, ranks regions by momentum, and aggregates a headline index.

The public version focuses on trend visibility and comparative rankings. A future premium layer can add forecasts, alerts, downloads, expanded region coverage, and blended indicators.
